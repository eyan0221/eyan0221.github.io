Git Tips

Sep 5th, 2014 10:04 pm

  1. Search
git log --author=
git branch --contains 
git log -S"key word"
  1. Change Author
git commit --amend --author=
  1. Rebase
git rebase -i HEAD~
edit
git commit --amend
git rebase --continue
  1. Reset
git reset --hard 
git reset --soft @{1}
git reset --hard HEAD
git reset --hard HEAD^
  1. Remote
git branch -r
git remote show origin
  1. Find commit list which contains 
git show `git log --oneline  | cut -c 1-8` | egrep -rs "|commit"
  1. Diff
git log --oneline ..
git log --oneline ..
